This is a simple multi-page website created using only HTML.  
The purpose of this project is to learn how to structure a website using semantic HTML elements.

## Pages Included

- Homepage
- Projects
- Articles
- Contact Page

## Features

- Semantic HTML structure
- Navigation bar on all pages
- Multiple pages connected with links
- Basic SEO meta tags
- Contact form with fields:
  - Name
  - Email
  - Message

## Technologies Used

- HTML5

## How to Run the Project

1. Download or clone the repository.
2. Open the project folder.
3. Double click on `index.html` file.
4. The website will open in your browser.
